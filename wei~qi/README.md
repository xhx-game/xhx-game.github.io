# 围棋游戏

一个基于HTML5、JavaScript和CSS3开发的在线围棋游戏，支持人机对战和AI强度调整。

## 功能特点

- **完整的围棋规则**：支持19路、13路、9路棋盘，包含吃子、提子、劫争等规则
- **AI对战系统**：提供可调节强度的AI对手（1-10级）
- **直观的用户界面**：清晰的棋盘显示和游戏信息
- **灵活的游戏控制**：新游戏、悔棋等功能
- **响应式设计**：适配不同屏幕尺寸

## 如何使用

1. 访问游戏页面
2. 选择棋盘大小（19路、13路或9路）
3. 调整黑方和白方的AI强度（1-10）
4. 勾选"使用AI"复选框启用AI对战
5. 点击棋盘落子开始游戏
6. 使用左侧的"新游戏"按钮重新开始
7. 使用右侧的"悔棋"按钮撤销上一步

## AI实现原理

### AI强度等级

- **1-3级**：随机落子，适合初学者
- **4-7级**：基于位置评估的策略性落子
- **8-10级**：使用Minimax算法的深度搜索

### 核心算法

1. **位置评估**：考虑中心位置、边角、连接性、吃子潜力等因素
2. **Minimax搜索**：
   - 8级：3层深度
   - 10级：4层深度
3. **Alpha-Beta剪枝**：优化搜索效率

## 技术栈

- **HTML5**：页面结构和Canvas绘图
- **JavaScript**：游戏逻辑和AI算法
- **CSS3**：样式设计和响应式布局
- **Node.js**：本地服务器

## 如何运行

### 本地运行

1. 确保安装了Node.js
2. 进入项目目录
3. 启动本地服务器：
   ```bash
   node server.js
   ```
4. 在浏览器中访问：http://localhost:8080

### 直接运行

可以直接在浏览器中打开`index.html`文件，但某些功能可能受限。

## 开发说明

### 主要类和方法

- **GoGame类**：游戏核心类
  - `constructor()`：初始化游戏
  - `makeMove(x, y)`：落子
  - `isValidMove(x, y)`：验证落子有效性
  - `getAIMove()`：AI决策
  - `minimax()`：AI搜索算法

### AI决策流程

1. 收集所有可能的落子位置
2. 根据AI强度选择策略
3. 进行位置评估或深度搜索
4. 返回最佳落子位置

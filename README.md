# XHX Games 游戏合集

欢迎来到 XHX Games 游戏库！这里汇集了所有已开发的小游戏。
您可以直接点击下方的链接开始游玩。

**项目主页**: [https://xhx-game.github.io](https://xhx-game.github.io)

## 🎮 游戏列表 (Game List)

### 1. Delta Pilot (三角洲飞行员)
一个霓虹风格的太空射击游戏。控制你的三角战机，消灭不断来袭的敌人！
- **试玩地址**: [https://xhx-game.github.io/delta_pilot/](https://xhx-game.github.io/delta_pilot/)

### 2. Snake (霓虹贪吃蛇)
经典贪吃蛇游戏的现代重制版，拥有炫酷的霓虹视觉效果和流畅的操作体验。
- **试玩地址**: [https://xhx-game.github.io/snake/](https://xhx-game.github.io/snake/)

### 3. Peacekeeper Lite (和平卫士 Lite)
一款俯视视角的射击与生存游戏，包含丰富的武器系统。
- **试玩地址**: [https://xhx-game.github.io/peacekeeper_lite/](https://xhx-game.github.io/peacekeeper_lite/)

### 4. Shooter Game (经典射击)
一个标准的基础射击游戏原型。
- **试玩地址**: [https://xhx-game.github.io/shooter_game/](https://xhx-game.github.io/shooter_game/)

### 5. Shopping Game (买菜游戏)
一个模拟买菜的休闲游戏。玩家需要根据每日任务，在菜市场购买指定的蔬菜，合理管理金钱并完成任务！
- **试玩地址**: [https://xhx-game.github.io/shopping_game/](https://xhx-game.github.io/shopping_game/)

### 6. Treasure War (宝物大战)
一款飞机射击类游戏，玩家控制飞机左右移动，射击敌人并收集宝物。包含生命值系统、关卡升级和多种奖励机制。
- **试玩地址**: 本地运行 (打开 treasure_war/index.html)

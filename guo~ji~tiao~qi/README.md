# 中国跳棋

一个基于HTML5、JavaScript和CSS3开发的在线中国跳棋游戏，支持三人对战和完整的游戏规则。

## 功能特点

- **完整的中国跳棋规则**：支持三人对战，包含相邻移动和跳跃移动
- **直观的用户界面**：清晰的棋盘显示和游戏信息
- **灵活的游戏控制**：新游戏、悔棋、重置游戏等功能
- **响应式设计**：适配不同屏幕尺寸
- **美观的视觉效果**：六边形网格和彩色棋子

## 如何使用

1. 访问游戏页面（index.html）
2. 点击红方棋子开始游戏
3. 点击目标位置落子
4. 依次轮到蓝方和绿方行动
5. 先将所有棋子移动到对方三角区域的玩家获胜

## 游戏规则

- 棋子可以向相邻的空位移动
- 棋子可以跳过相邻的棋子（自己或对手的）到达对面的空位
- 每个玩家有10个棋子
- 红方起始位置：顶部三角区域
- 蓝方起始位置：底部三角区域
- 绿方起始位置：右侧三角区域
- 获胜条件：将所有10个棋子移动到对方的三角区域

可以直接在浏览器中打开`index.html`文件开始游戏。

## 开发说明

### 主要类和方法

- **ChineseCheckersGame类**：游戏核心类
  - `constructor()`：初始化游戏
  - `initBoard()`：初始化棋盘
  - `makeMove(fromX, fromY, toX, toY)`：移动棋子
  - `isValidMove(fromX, fromY, toX, toY)`：验证移动有效性
  - `checkWin()`：检查游戏胜负
  - `drawBoard()`：绘制棋盘和棋子
